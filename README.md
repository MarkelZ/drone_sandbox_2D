# Drone Sandbox 2D

## Description
This is a sandbox for making flying machines in 2D.

## Demo

https://user-images.githubusercontent.com/60424316/228384568-49955af6-a7d5-44f6-b072-991a284612f1.mp4

## Dependencies
- Python 3.
- Pygame 2.3.0+

## Running the sandbox
Run the following command:
```sh
cd src/sandbox/
python3 main.py
```
